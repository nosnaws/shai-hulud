## instructions

### dependencies

- yarn 1.22
- node.js 16

get the rest
```
yarn install  
```

### running tests

```
yarn test
```

### running the sssssserversnake
compile typescript
```
yarn build 
```

run server
```
PORT=8080 yarn start
```
